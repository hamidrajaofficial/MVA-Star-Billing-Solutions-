# Global UI Rules
- When creating feature or service cards, always place the Icon and the Heading (`h3`/`h4`) in a single row (using `flex items-center gap-4`) rather than stacking them vertically.
- Icons should always be 300% relevant to the content (use high-context Lucide icons).
- Maintain a consistent vertical spacing (padding) of 30px on top and bottom of each section (`py-[30px]`).
- For alternating sections (sections that are not white), use the background color `#F4F9F9` (represented by the tailwind class `bg-section-bg`). Always check background contrast especially after white sections.
